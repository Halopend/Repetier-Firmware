This is a version of repetier firmware ported to run on the rostock max 3d printer by SeeMeCNC. It supports and is preconfigured to enable LCD controller support.

Install

First you'll need to download the firmware(!). Depending on your priorities, you can either just download the zip file above and use that or use git (which you'll have to install) to clone the repository using:

cd folder/where/you/want/to/install/
git clone https://github.com/repetier/Repetier-Firmware.git

Updating consist of:

cd path/to/repetier-firmware/
git pull

Setup

You'll need to download the arduino software from here to compile and upload the Firmware to your board. Use it to open the file repetier-firmware/Reptier.pde. In the arduino app, go to tools>board>arduino mega 2560 and make sure that it's checked. with your board connected over USB, hit upload and you should be good to go.

With EEPROM support enabled (which is the default), you need to intialize the settings the first time you run the firmware using the M-Codes:

M502
M500
This loads the default settings from the firmware to memory and then writes these settings to the EEPROM. Without doing this step, all your firmware settings will be at the mercy of whatever previous firmwares wrote to EEPROM before and your system will behave completely unpredictably.
